1. Qui sommes-nous ?

MémoVie est une application Android développée par ProtéFinance, éditeur français de logiciels de gestion personnelle et professionnelle.

Contact : contact@protefinance.fr

2. Données collectées

🔒 MémoVie ne collecte aucune donnée personnelle sur des serveurs externes.
Toutes vos données restent sur votre appareil, chiffrées avec AES-256-GCM.

Les données suivantes sont stockées uniquement sur votre appareil :

  • Vos notes, promesses, dettes, rappels et entrées de santé
  • Votre profil (pseudo, email, prénom, nom)
  • Vos contacts de l'annuaire personnel
  • Vos préférences d'application (thème, notifications)

3. Chiffrement et sécurité

Toutes les données sont chiffrées avec AES-256-GCM via le Keystore Android. La clé de chiffrement est dérivée de votre PIN personnel avec PBKDF2 (100 000 itérations).

L'accès à l'application est protégé par :

  • Code PIN à 4 chiffres
  • Authentification biométrique (empreinte digitale, reconnaissance faciale)

4. Intelligence Artificielle (optionnelle)

Si vous choisissez d'activer les fonctions IA, votre clé API Anthropic est stockée localement sur votre appareil, chiffrée. Les requêtes IA sont envoyées directement à l'API Anthropic (anthropic.com).

Consultez la politique de confidentialité d'Anthropic pour les données traitées lors des appels API. L'activation de l'IA est entièrement optionnelle.

5. Permissions Android

  • Microphone : saisie vocale, uniquement à votre demande explicite
  • Notifications : rappels programmés par vos soins
  • Biométrie : déverrouillage sécurisé de l'application
  • Stockage : export de vos données (.mvault)

6. Partage de données

MémoVie ne partage aucune donnée avec des tiers.
Aucune publicité. Aucun tracking. Aucune analytics.

7. Suppression de vos données

Vous pouvez supprimer toutes vos données à tout moment :

  • Dans l'app : Réglages → Données → Supprimer toutes les données
  • Désinstallation : la désinstallation de l'application supprime toutes les données locales

8. Droits RGPD

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès à vos données (export .mvault dans Réglages)
  • Droit de rectification (modification directe dans l'app)
  • Droit à l'effacement (suppression dans Réglages)
  • Droit à la portabilité (export .mvault chiffré)

Pour exercer ces droits : contact@protefinance.fr

9. Modifications

Cette politique peut être mise à jour. La date de dernière modification est indiquée en haut de cette page. Les modifications importantes seront notifiées dans l'application.
